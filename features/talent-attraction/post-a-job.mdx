Follow the guided flow to create a job post that leverages skills-based matching to find the right candidates.

## Job posting flow

<Steps>
  <Step title="General information">
    Define the basic details of the job.

    <Frame>
      ![Image](/images/image-47.png)
    </Frame>
    Fill in the following fields:

    | Field               | Description                                                          |
    | :------------------ | :------------------------------------------------------------------- |
    | **Channels**        | Select e2i jobseekers, Your employees, and/or JobTech talent databse |
    | **Employer**        | The hiring company                                                   |
    | **Hiring manager**  | Person overseeing the role                                           |
    | **Job owner**       | Person accountable for the posting                                   |
    | **Job title**       | Title of the position                                                |
    | **Employment type** | Full-time, Part-time, Contract, etc.                                 |
    | **Job type**        | Classification of the role                                           |
    | **Vacancies**       | Number of openings                                                   |
    | **Workplace type**  | Remote, Hybrid, or On-site                                           |
    | **Salary range**    | Minimum and maximum compensation                                     |
  </Step>
  <Step title="Job description, job requirements, and required skills">
    Provide a detailed description of the role and its responsibilities.

    <Frame>
      ![Image](/images/image-48.png)
    </Frame>
    Write a clear overview covering:

    - Key responsibilities
    - Day-to-day activities
    - Team structure and reporting line
  </Step>
  <Step title="Requirements">
    Define some hiring and academic requirements.

    <Frame>
      ![Image](/images/image-49.png)
    </Frame>
  </Step>
  <Step title="Additional details">
    Set experience requirements and other preferences.

    <Frame>
      ![Image](/images/image-50.png)
    </Frame>
    - **Experience** — Minimum years required
    - **Commitment** — Hours per day or week
    - **Academic qualifications** — Minimum education level
    - **Job opening period** — Start and expiry dates
    - **Visibility settings** — Options such as "Open to PWD"
  </Step>
  <Step title="Review and post">
    Review all details before submitting.

    - Confirm all sections are complete
    - Click **Post** to submit the job post.
    - If you are e2i Partner Employer, you might need to wait for the e2i Job Owner approval.
  </Step>
</Steps>

## Bulk upload

For posting multiple jobs at once, use the bulk upload feature. [Download template here.](https://uat.betalented.ai/download/ta-jobs-bulk-upload-template/1)

<Steps>
  <Step title="Download the template">
    Download the CSV template from the Talent Attraction dashboard.
  </Step>
  <Step title="Fill in the template">
    Enter job details for each position in the template file.
  </Step>
  <Step title="Upload the file">
    Upload the completed file. The system will map each column to the corresponding field.
  </Step>
  <Step title="Review and confirm">
    Review the imported data for accuracy, then confirm to create all job posts.
  </Step>
</Steps>

<Warning>
  Ensure all required fields are filled in the template. Incomplete rows may be rejected during import.
</Warning>
