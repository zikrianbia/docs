The User Directory is a central directory listing all employees and their skill profiles. Use it to discover experts, view colleague profiles, and find the right people for projects or collaboration.

## Browsing the directory

When you open the [User Directory](https://app.betalented.ai/user-directory), you will see a list of all employees in the organisation.

## Employee skill profile

Click on any employee to view their detailed skill profile.

The profile includes:

- **Skills list** — All skills in the employee's profile
- **Proficiency levels** — Each skill is displayed with a proficiency rating ranging from **Basic** to **Expert**

<Tip>
  Use the User Directory to find subject matter experts when you need guidance on a specific topic, or to discover potential mentors for your development plan.
</Tip>
