The Analytics Dashboard provides an overview of insights, trends, and metrics to help administrators track performance and make informed decisions. It is divided into two main views: **Overview** and **Skills**.

## Overview

The Overview tab displays key metrics about your organisation's platform usage and workforce data.

[Upload Image 1 here]

### Key metrics

| Metric | Description |
|:---|:---|
| **Headcount** | Total number of employees in the system |
| **Onboarded Headcount** | Number of employees who have completed onboarding |
| **Unique Sign Ins** | Distinct users who signed in over the past 12 months |
| **Unique Active Users** | Users actively engaging with the platform over 12 months |
